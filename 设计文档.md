## 二、选题分析

### 2.1 赛道定位

选题打"双相情感障碍伴侣视角"，报名小红花赛道（公益游戏）。同时命中：
- **赛道一（公益）**：精神健康是明确的公益议题，双相障碍长期被污名化
- **赛道三（叙事创新）**：AI NPC 状态切换 + 机器人身份认知是结构性创新

### 2.2 核心叙事钩子

**表层叙事**：你与双相障碍伴侣阿林的日常陪伴，识别她的状态变化，做出影响关系的选择。

**深层叙事**（Demo 埋暗示，完整版揭示）：你（莉雅）是一台陪伴型机器人，因保护阿林被棒球棍击中导致程序错乱，忘记了自己是机器人，以为自己是阿林的伴侣。

**双重隐喻**：
- 阿林的双相障碍 → "我不知道下一秒的自己是谁"
- 莉雅的程序错乱 → "我不知道自己到底是谁"
- 两个角色在"自我认知混乱"这一主题上形成镜像

### 2.3 公益价值传达

- 展示双相障碍患者的真实状态（躁狂/抑郁/平稳）
- 通过互动让玩家体验"识别"的过程
- Demo 结尾展示公益信息与求助渠道
- 章节间可插入科普小知识（可选）

## 三、角色设定

### 3.1 阿林 (Alin) — NPC

| 属性 | 内容 |
|------|------|
| 全名 | 林莉雅 |
| 性别 | 女 |
| 年龄 | 25 岁 |
| 身份 | 双相情感障碍患者 |
| 背景 | 被前男友山姆伤害，在清醒状态下购买陪伴机器人 |
| 与莉雅关系 | 购买者/被陪伴者，给机器人取了自己的名字（莉雅 = 林莉雅） |

**AI 状态机**：

| 状态 | 语言特征 | 行为特征 | 视觉色调 |
|------|----------|----------|----------|
| 躁狂期 | 语速极快、话题跳跃、宏大计划、易怒 | 不睡觉、疯狂工作、冲动消费倾向 | 高饱和橙红/亮黄 |
| 抑郁期 | 沉默、自责、回避、语言迟缓 | 无法起床、社交退缩、身体不适 | 低饱和蓝灰/暗蓝 |
| 平稳期 | 正常交流、可谈病情、有自省 | 规律生活、服药依从、社交可能 | 暖橙/奶油/浅灰 |
| 混合发作 | 抑郁底色 + 躁狂能量 | 最危险，高自杀风险 | 冷暖冲突 |

## 四、Demo 剧本

### 4.1 整体结构

```
序章：袭击（~2 分钟）
├── 山姆闯入 → 莉雅保护阿林
├── 被棒球棍击中 → 程序错乱
├── 夺棍击退山姆（力量暗示）
├── 记忆混乱，忘记自己是机器人
└── [自由对话1：乱码模式]

第一幕：躁狂爆发（~1.5 分钟）
├── 几天后深夜，阿林疯狂工作
├── 展示躁狂期语言和行为模式
└── 固定对话 + 关键选择分支

第二幕：闪回·平稳期（~1.5 分钟）
├── 回忆"初识"场景
├── 同一角色的温柔对话（巨大反差）
└── 埋暗示（名字由来、孤独感）

第三幕：抉择（~1.5 分钟）
├── 回到现在，阿林情绪崩溃
├── [自由对话2：正常模式 2-3轮]
├── 关键选择：如何回应她的崩溃
└── 三种即时反应，引向尾声

尾声（~15 秒）
├── 阿林在沙发上睡着
├── 莉雅静静坐在旁边，窗外2046年城市夜景
├── 莉雅看向自己的手，眼神困惑
└── 画外音 + 公益信息 + 未完待续
```

### 6.3 自由对话系统

#### 两种模式

| 模式 | 触发位置 | 表现 | 目的 |
|------|----------|------|------|
| 乱码模式 | 序章结束后 | 玩家输入→显示乱码→阿林无回复 | 展示程序受损 |
| 正常模式 | 第三幕抉择前 | 玩家输入→正常显示→阿林正常回复 | 情感积累 |

#### 乱码实现

```javascript
function scrambleText(text) {
  const chars = text.split('');
  const result = [];
  for (let i = 0; i < chars.length; i++) {
    if (Math.random() < 0.4) {
      result.push('▯');
    } else if (Math.random() < 0.2) {
      result.push('...');
    } else {
      result.push(chars[i]);
    }
  }
  return result.join('');
}
```

### 9.2 选择系统

- **选项按钮**：居中排列，半透明背景 + 柔光边框
- **悬停效果**：边框发光增强

### 9.3 自由对话输入

- **输入框**：底部弹出，圆角，半透明背景
- **发送按钮**：右侧箭头图标
- **乱码显示**：损坏文字使用故障字体效果 + 轻微抖动动画

### 9.4 转场效果

- **场景切换**：淡入淡出（0.5秒）
- **闪回**：暖光扩散过渡
- **程序错乱**：画面闪烁 + RGB 分离效果
- **黑屏过渡**：纯黑 + 字幕居中淡入

### 9.5 字体

- **正文**：思源黑体 / Noto Sans SC（无衬线，可读性好）
- **乱码**：等宽字体（Courier New / Fira Code）
- **标题**：粗体，带发光效果
