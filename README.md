## 简介

2046年，你扮演**莉雅**，与伴侣**阿林**共同生活。阿林是一位双相情感障碍患者——她的世界时而光芒万丈，时而沉入深渊。在一次暴力袭击中，你的头部遭到重击，某种深层程序因此错乱。你忘记了自己是谁，只记得一件事：陪伴阿林。

游戏融合**对话选择**与 **AI 驱动的自由对话**——阿林由 DeepSeek-v4-pro 大模型实时驱动，你的每一句话都会得到即兴、真实的回应。通过识别她的躁狂期、抑郁期与平稳期，做出影响她命运的选择。

## 快速开始

```
直接打开浏览器访问:
https://pxlm570.github.io/the-switch/
```

无需安装、无需下载、无需注册。

**操作方式**：
- 点击对话框 / 按空格键 / 回车 → 推进对话
- 点击选项按钮 → 做出选择
- 输入文字 + 发送 → 自由对话
- 左上角 ⚙ → 保存/读取/设置

## 项目结构

```
├── index.html          # 主页面
├── css/style.css       # 赛博柔光风样式
├── js/
│   ├── engine.js       # 视觉小说核心引擎
│   ├── script.js       # 完整剧本数据
│   ├── llm.js          # DeepSeek API + fallback
│   ├── state.js        # 状态管理
│   ├── save.js         # 存档系统
│   └── audio.js        # Web Audio 音效合成
└── assets/
    ├── bg/             # 4 张场景背景
    └── chars/          # 5 张角色立绘
```

## 核心特色

- **AI 原生叙事**：NPC 由 LLM 实时驱动，四套状态 Prompt 切换，每局对话不同
- **双重隐喻**：阿林的"我不知道下一秒的自己是谁" × 莉雅的"我不知道自己是谁"
- **视觉即叙事**：冷暖色调随角色情绪实时变化
- **公益价值**：关注双相情感障碍，增进公众理解
- **全 AI 创作**：美术、音效、NPC 对话、引擎代码均由 AI 辅助生成
