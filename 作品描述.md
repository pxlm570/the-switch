# 《看不见的开关》The Switch — 作品描述

## 一句话简介

> 2046年，你与双相障碍伴侣共同生活。你的每一个选择，都可能改变她的命运——而你，甚至不知道自己是谁。

---

## 作品背景

2046年，AI已深度融入人类社会。你扮演**莉雅**，与伴侣**阿林**共同生活在近未来的城市公寓中。

阿林是一位双相情感障碍患者。她的世界，时而光芒万丈，时而沉入深渊。躁狂期里，她能通宵写出三十二页商业计划书；抑郁期里，她连从沙发走到门口的力气都没有。

而你的存在本身就是一个谜——在一次暴力袭击中，你被棒球棍击中头部，某种深层程序因此错乱。你忘记了自己是谁，只知道一件事：**陪伴阿林**。

---

## 核心玩法

### 对话选择 × 状态识别

阿林的状态会在**躁狂期**、**抑郁期**和**平稳期**之间切换。作为她最亲近的人，你需要通过对话线索识别她的当前状态，并做出恰当的回应——错误回应可能加深她的危机，正确陪伴则能帮她渡过难关。

### AI 驱动的自由对话

在关键情节中，你可以**自由输入想说的话**，阿林会通过 DeepSeek-v4-pro 大模型给出真实、即兴的回复。不是预设脚本，而是每一局都不同的对话体验。AI 会根据她当前的情绪状态调整语气、用词和微表情，让每一次交流都不可复制。

### 叙事深度递进

Demo 包含序章、三幕正剧和尾声，约 6-8 分钟体验。表面上这是一个关于陪伴与选择的故事，但字里行间埋藏着更深层的谜题——莉雅到底是谁？那双没有伤痕的手、那段闪烁的代码碎片、那句"你的名字是我取的"——都在暗示一个你尚未知晓的真相。

## 核心亮点

| 亮点 | 说明 |
|------|------|
| **AI 原生叙事** | 非预设对话树，而是 LLM 驱动的即兴角色扮演。NPC 拥有完整的状态机（躁狂/抑郁/平稳/混合），对话表现随状态实时变化 |
| **公益价值** | 关注双相情感障碍群体，通过互动体验增进公众理解。Demo 结尾展示真实求助信息 |
| **双重隐喻** | 阿林的"我不知道下一秒的自己是谁"与莉雅的"我不知道自己到底是谁"形成镜像叙事，探讨自我认知的普世命题 |
| **赛博柔光美学** | 自研扁平色块+柔和光效的美术风格，冷暖色调随角色情绪同步变化，视觉即叙事 |
| **全 AI 创作** | 从美术资源（10张场景+角色立绘）、音效（Web Audio 合成）、到 NPC 对话，全部由 AI 辅助生成 |
| **零依赖部署** | 纯静态 Web 页面，打开浏览器即玩，无需安装 |

---

## 技术架构

- **引擎**：自研纯 Web 视觉小说引擎（HTML/CSS/JS）
- **AI 对话**：DeepSeek-v4-pro API，JSON 结构化输出，含多状态 system prompt 与离线 fallback 对话池
- **美术**：AI 生成赛博柔光风场景背景 + 角色立绘
- **音效**：Web Audio API 实时合成（破门/碎裂/故障/打字机/门铃）
- **存档**：localStorage 5 槽位，支持快速存档/读档
