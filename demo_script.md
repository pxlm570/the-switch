| 时间段 | 内容 | 时长 |

## 完整解说词

### 开场 (0:00-0:20)

```
[画面：标题画面，城市夜景背景，"看不见的开关"标题发光]

2046年，人工智能已融入人类生活的每个角落。
但有些问题，技术无法回答。

你扮演莉雅，与阿林共同生活。
阿林是你的伴侣，也是一位双相情感障碍患者。
她的世界，时而光芒万丈，时而沉入深渊。
而你的每一个选择，都可能改变她的命运。
```

### 序章 (0:20-1:00)

```
[画面：序章日常场景 → 阿林在沙发上，暖橙色灯光]

一切从一个普通的傍晚开始。

[画面：门被踹开特效 + 屏幕震动 + 碎裂音效]

直到阿林的前男友山姆闯入。
为了保护阿林，你挡在了前面。
棒球棍击中了你的头部。

[画面：屏幕全白 → glitch 代码碎片闪烁]

画面闪烁。代码碎片从视野边缘划过。
系统受损。记忆索引损坏。

[画面：夺棍，山姆惊恐逃出]

你夺下了棒球棍，击退了山姆。
但有什么东西...不一样了。
```

### 第一章：躁狂爆发 (1:00-1:40)

```
[画面：深夜客厅，高饱和暖色，阿林在桌前疯狂工作]

几天后的深夜。
阿林没有睡——她已经连续工作了二十个小时。

[画面：阿林快速说话，语速极快，情绪亢奋]

"我想到一个改变世界的计划！"
"天才不需要睡觉！"

这是躁狂期——双相障碍的一面。
能量过载，计划宏大，情绪高涨。

[画面：选项出现，玩家点击]

你的回应，会影响她的状态。
是让她冷静，还是倾听她的热情？
```

### 第二章：闪回·平稳期 (1:40-2:10)

```
[画面：咖啡馆，暖橙色柔和光线，阿林微笑]

但这不是她本来的样子。

[画面：阿林捧着咖啡，温柔对话]

"你的名字——莉雅——是我取的。"
"用了我自己的名字。林莉雅。"

在平稳的日子里，她温柔、自省、会笑。
同一双手，既能握紧咖啡杯，
也会在深夜发抖。

[画面：色调从暖转冷，场景过渡回深夜客厅]

这种反差，就是双相障碍的真实面貌。
```

### 第三章：抉择 (2:10-2:40)

```
[画面：阿林在沙发上抱头，情绪崩溃]

"我是不是永远都这样了？"
"一会儿觉得自己能改变世界，一会儿连站起来的力气都没有。"

[画面：自由对话输入框弹出，玩家输入文字]

这是游戏的核心机制——自由对话。
你输入的每一个字，都会得到阿林的真实回应。
不是预设脚本，而是AI驱动的即兴对话。

[画面：关键选择出现]

"你会离开我吗？"

你的答案，决定故事的走向。
```

### 尾声 + 公益信息 (2:40-3:00)

```
[画面：阿林在沙发上睡着，莉雅坐在旁边，窗外城市夜景]

[画面：黑屏，字幕浮现]

双相情感障碍影响着全球约4500万人。
识别，是帮助的第一步。

《看不见的开关》
完整版，即将呈现。

[画面：公益信息 + 求助渠道]
```

## 录制操作指南

1. 打开 `index.html`（本地服务器或部署后的链接）
2. 分辨率设置为 1920x1080
3. 按脚本顺序操作游戏，每个场景停留对应时长
4. 录制时打开系统声音，确保音效被收录
5. 后期叠加解说词（可用 AI TTS 生成配音）

## 关键操作节点

| 时间点 | 操作 |
|--------|------|
| 0:05 | 点击"新游戏" |
| 0:20 | 序章对话逐句点击推进 |
| 0:35 | 序章选择 → 点击"挡在阿林面前" |
| 0:45 | 序章乱码自由对话 → 输入任意文字发送 |
| 1:05 | 第一章躁狂对话推进 |
| 1:25 | 第一章选择 → 点击任一选项 |
| 1:45 | 第二章闪回咖啡馆对话推进 |
| 2:00 | 第二章选择 → 点击任一选项 |
| 2:15 | 第三章自由对话 → 输入一句安慰的话发送 |
| 2:25 | 第三章自由对话 → 再发送一句 |
| 2:30 | 第三章关键选择 → 点击任一选项 |
| 2:40 | 尾声自动播放 |
